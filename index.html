Автостопом по CSS Style Guide Галактике
Требования к приложениям растут, а с ними увеличиваются и объемы кода, которые разработчики пишут ежедневно. Неважно, насколько большой по объему проект вы пишите: личный блог или это новый "убийца" Фейсбук. Чтобы этот проект не превратился в свалку, вам нужно руководство по написанию кода. Если в отношении JavaScript на большинстве проектов дела обстоят более-менее, то в отношении CSS ситуация вообще плачевна. В этом докладе мы рассмотрим что такое CSS Style Guide, как он выглядит, что туда должно входить, кто его должен писать и другие не менее важные пункты.
