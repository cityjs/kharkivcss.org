KharkivCSS #4

Харків, Fabrika.space, вул. Благовіщенська, 1
20 Квітня, 2019
KharkivCSS — спільнота для Frontend/Backend розробників та дизайнерів, котрі хочуть дізнатися більше про нові і актуальні технологічні стандарти Всесвітньої павутини, обмінятися досвідом і поспілкуватися з однодумцями.

Програма

Реєстрація

10:00

Lea Verou

[Greece]

Автор бестселера передової книги CSS Secrets. Працювала розробником-адвокатом W3C. Має давню пристрасть до відкритих веб-стандартів і є однією з небагатьох запрошених експертів у CSS Working Group.

Лія запустила кілька популярних open source проектів та веб-додатків, таких як Mavo, Prism та -prefix-free, веде технічний блог lea.verou.me. Має ступінь магістра в галузі комп'ютерних наук MIT. Незважаючи на її наукові заняття в області комп'ютерних наук, Лія з тих, хто полюбляє код і дизайн однаково.

Зараз Лія займається дослідженнями того, як спростити веб-програмування в MIT CSAIL.

11:00

Chris Lilley

[France]
Технічний директор World Wide Web Consortium(W3C). Будучи "батьком" SVG, він також є співавтором PNG, був співредактором CSS2, очолював групу, яка розробляла @font-face, та спільно розробляв WOFF. Також, був членом W3C Technical Architecture Group. В даний час працює на CSS рiвнях 3/4/5, Web Audio та WOFF2.

Антон Немцев

[Україна]

З 2000 по 2016 р.р був незалежним розробником. Продався корпораціям.
Засновник і редактор електронного журналу Frontender Magazine. Представник організації Web Standards в Україні.

Доповідач на міжнародних і локальних конференціях веб-розробників.
Експерт на чемпіонаті UA Web Challenge.
Володіє повним стеком технологій, необхідних для розробки веб-проектів.

WCAG: изучаем основы веб-доступности на практике

[Ru]

WCAG является наиболее широко используемым во всём мире стандартом по обеспечению web-accessibility. Мы поговорим о практической стороне реализации веб-инклюзии. Посмотрим, как сделать интерфейс удобным для всех, используя практики WCAG, рассмотрим некоторые неочевидные решения и немножко послушаем музыку.

Дмитро Попов

[Україна]

Більше 10 років займається питаннями доступності веб-технологій для людей з порушенням зору. Працював над підвищенням доступності сайту Укрзалізниці. Тестував на доступність сайти ЦОВ в рамках проекту "Детектор веб-доступності".

Денис Яровий

[Україна]
